GCP Vertex AI 서비스 계정 키 발급 순서

1
GCP 콘솔 접속
  • console.cloud.google.com 접속 후 로그인
  • 무료 체험 크레딧($300)이 활성화된 계정으로 접속
2
Vertex AI API 활성화
  • 검색창 → Vertex AI 검색
  • Vertex AI API → 사용 클릭
3
서비스 계정 생성
  • 검색창 → IAM 및 관리자 → 서비스 계정
  • 서비스 계정 만들기 클릭
  • 역할: Vertex AI 사용자 선택
4
JSON 키 다운로드
  • 만든 서비스 계정 클릭 → 키 탭
  • 키 추가 → 새 키 만들기 → JSON 선택
  • 다운로드된 .json 파일을 안전한 곳에 보관
5
앱에 등록
  • API Key 설정 클릭 → Vertex AI 섹션
  • 찾아보기로 JSON 파일 선택 → OK
  • Gemini API 키란은 비워도 됩니다
※ Vertex AI JSON 키가 등록되면 Gemini API 키 대신 GCP 크레딧(₩441,614)으로 자동 과금됩니다.
※ 서비스 계정 JSON 파일은 외부에 유출되지 않도록 주의하세요.
